Sleeper Fantasy Football Exposure Tool Tools used: HTML, JavaScript, CSS, Sleeper API, AnyChart

This is a tool for users of the popular Fantasy Sports application, Sleeper. The user can input their Sleeper username, and the tool uses the Sleeper API and AnyChart to create a bar graph of their most frequently owned players throughout all of their Fantasy Football leagues.
